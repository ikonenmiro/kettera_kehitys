# Ketterä kehitys
1. Forkkaa tämä repositorio omaan GitHubiisi
2. Kloonaa tämän jälkeen omalle työasemallesi
3. Luo kettera_kehitys hakemistoon hakemisto omalla nimelläsi esim. *etunimi.sukunimi*
4. Luo *etunimi.sukunimi* hakemistoon tiedosto *kettera_kehitys.md*
5. Lisää *kettera_kehitys.md* tiedostoon vastaukset seuraaviin kysymyksiin
- Mitä ketterällä kehityksellä tarkoitetaan?
- Mitä menetelmää käytettiin ennen ketterää kehitystä?
- Miksi ketterään kehitykseen on siirrytty?
- Kerro lyhyesti agilesta?
- Mitä ovat Scrum ja Kanban?
- Miten Scrumin ja Kanbanin olennainen ero?
- Miten Jira liittyy ketterään kehitykseen?
- Miten Git edistää ketterää kehitystä?

6. Kun olet vastannut kysymyksiin omalla työasemalla olevassa kloonatussa hakemistossa, puske vastaukset (ja tiedosto) omaan GitHub ympäristöösi
7. Tämän jälkeen tee *pull request* alkuperäiseen repositorioon.
